## C语言实现的扫雷游戏 

#### 硬件和系统
- CPU: IMX6ULL
- MEMORY: 512MB
- EMMC: 8.00GB
- RESOLUTION: 1024 x 600
- SYSTEM: Linux 4.1.15

#### 基本功能：

1. 拥有一个主菜单功能界面 能够通过点击区域来进入不同功能 
2. 拥有不同的难度等级 通过点击选择 `[√]`
    难度等级为扫雷区域大小不同 `[√]`
3. 插旗功能 `[√]`
4. 得分功能的实现和显示 `[√] 剩余地雷数量显示`
5. 成功与失败的提示 `[√] 成功或失败时全部地雷显示，并且计时暂停`
6. 可以重新开始游戏 `[√]`

#### 额外功能： 

1. 计时功能的实现和显示 `[√]`
2. 用户账户和密码机制 
3. 用户数据记录 如 胜场和败场记录 
4. 用户胜场排行榜(可查询前10选手数据功能 最好能看自己的排名)
